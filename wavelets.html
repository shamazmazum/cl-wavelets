Currently these wavelets are supported:
  1. Haar wavelet. Call transform functions with :wavelet :haar argument to use this wavelet.
  2. CDF (2,2) Call transform functions with :wavelet :cdf-2-2 argument to use this wavelet.
  3. CDF (4,2) Call transform functions with :wavelet :cdf-4-2 argument to use this wavelet.
